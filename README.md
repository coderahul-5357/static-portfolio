# Endla Rahul - Personal Portfolio Landing Page

A modern, creative, and responsive personal portfolio landing page built with React, Tailwind CSS, and Framer Motion.

## 👨‍💻 About

This is my personal portfolio showcasing my journey as a Computer Science student specializing in AI/ML at MS Ramaiah University of Applied Sciences. The website features my education, skills, projects, and ways to get in touch.

## 🚀 Features

- **Responsive Design** - Fully responsive across all devices
- **Smooth Animations** - Beautiful animations using Framer Motion
- **Modern Styling** - Tailwind CSS for sleek, modern design
- **Multiple Sections**:
  - Hero/Welcome section
  - About Me
  - Education & Background
  - Skills & Technologies
  - Featured Projects
  - Contact Information

## 🛠️ Tech Stack

- **Frontend**: React 19
- **Build Tool**: Vite
- **Styling**: Tailwind CSS 4
- **Animations**: Framer Motion
- **Language**: JavaScript (JSX)

## 📋 Prerequisites

- Node.js (v16 or higher)
- npm or yarn

## 🚀 Getting Started

### Installation

1. Clone this repository:
```bash
git clone https://github.com/yourusername/landing-page.git
cd landing-page
```

2. Install dependencies:
```bash
npm install
```

### Development

Start the development server:
```bash
npm run dev
```

The site will be available at `http://localhost:5173/`

### Build

Create a production build:
```bash
npm run build
```

### Preview

Preview the production build:
```bash
npm run preview
```

## 📝 Customization

### Update Personal Information

Edit `src/components/Hero.jsx`, `src/components/About.jsx`, and other components to add your:
- Name and professional title
- Bio and interests
- Education details
- Projects and achievements
- Contact information

### Modify Colors

Edit `tailwind.config.js` to customize the color scheme:

```javascript
theme: {
  extend: {
    colors: {
      primary: '#6366f1',      // Change primary color
      secondary: '#ec4899',    // Change secondary color
    },
  },
},
```
